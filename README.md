# 📚 G&P Literatura Cristiana - Librería Online

¡Bienvenido/a al repositorio de **G&P Literatura Cristiana**, una librería virtual especializada en literatura cristiana para todos los públicos!

Este proyecto es una tienda en línea que permite visualizar productos sin necesidad de registrarse, pero requiere crear una cuenta para comprar. Incluye funciones de registro, inicio de sesión, carrito de compras y administración de clientes.

---

## 🌐 Características

- 🖥️ Interfaz responsiva con HTML5 + Bootstrap 5
- 🔐 Registro e inicio de sesión para clientes
- 🛒 Carrito de compras funcional
- 👤 Panel de administración para gestionar usuarios
- 🗂️ Base de datos en MySQL conectada con PHP (PDO)
- 💬 Categorías: Biblias, libros, diccionarios, comentarios, atlas, etc.

## ⚙️ Cómo ejecutar el proyecto

1. Clona el repositorio o descárgalo como `.zip`
2. Coloca la carpeta dentro de `C:\xampp\htdocs\`
3. Inicia **Apache** y **MySQL** desde XAMPP
4. Importa el archivo `.sql` con la base de datos (`libreriavirtual`)
5. Accede desde tu navegador:  

http://localhost/Formulario_Libreria_Online/Inicio.php
