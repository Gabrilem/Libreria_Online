# 📚 G&P Literatura Cristiana

**Palabras que dan vida**

Este proyecto es una aplicación web para una librería cristiana en línea. Permite a los usuarios registrarse, iniciar sesión, consultar su perfil y gestionar compras.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap
- PHP
- MySQL
- XAMPP
- JS

---

## 🚀 Funcionalidades principales

- Registro de clientes
- Inicio y cierre de sesión
- Perfil del cliente
- Panel de administración
- Carrito de compras (en desarrollo)

---

Abre servidor XAMPP
abre BD en myadminphp ADMIN en la parte del MYSQL port 3306
